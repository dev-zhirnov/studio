# Studio Site

A modern Next.js application for Zhirnov Studio.

## Tech Stack

- **Framework:** Next.js 14+
- **Language:** TypeScript
- **Styling:** CSS Modules
- **Deployment:** Docker + Docker Compose

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Docker

```bash
docker-compose up -d
```
